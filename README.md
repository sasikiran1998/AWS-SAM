# TIC-TAC-TOE
## Using HTML, CSS, JS.

### How to Run ?

* Clone the project.
* Open with VS Code.
* Open Live Server Extension and preview the project.
* OR
* Run with the help of Localhost using any supportive browsers like Chrome, Firefox, Microsoft Egde, or any secure browser.
* Play the Game... !!
